Alice and Bob play the following game.They choose a number N to play with.The runs are as follows :

1.Bob plays first and the two players alternate.

2.In his/her turn ,a player can subtract from N any prime number(including 1) less than N.The number thus obtained is the new N.

3.The person who cannot make a move in his/her turn loses the game.

Assuming both play optimally,who wins the game ?

Input format:

The first line contains the number of test cases T.Each of the next lines contains an integer N.

Output format:

Output T lines one for each test case,containing "ALICE" if Alice wins the game ,or "BOB" if Bob wins the game.

Example

Sample Input:
2
1
2


Sample Output:
ALICE
BOB


Constraints:
1 <= T <= 1000000
1 <= N <= 1000000000

Note : For the first test case, Bob cannot make any move and hence Alice wins the game. For the second test case, Bob subtracts 1 from N. Now, Alice cannot make a move and loses the game.

Given any n%4 != 1 and it's

kaq1 @ 18 Jul 2014 06:14 AM
Given any n%4 != 1 and it's your turn, you can make the other person's turn have n%4 == 1 by subtracting 1, 2, or 3. That person can't make your turn have n%4 == 1 since no prime is a multiple of 4. If you continue with forcing the other person to have n%4 == 1 on their turn, it'll soon decrease to n = 1 (since n always decreases) on their turn, so you win.

For these problems in

kaq1 @ 18 Jul 2014 06:19 AM
For these problems in general, try working with small cases. Start with n = 1 and work your way up (n++). For every n, write it on a piece of paper (or anywhere, doesn't really matter). Circle that value if you lose if it's your turn and n = that value. You lose if you can't send the other person to a value with a circle around it. Ex: If n = 1, you lose automatically so circle 1. If n = 2, you can subtract 1 and send the other person to n = 1. For n = 3 and 4, you can subtract 2 and 3 respectively. For n = 5, you can only subtract 1, 2, or 3, so you're forced to let the other person start with a winning number, making your number a losing number. For n = 6, 7, and 8, you can subtract 1, 2, or 3, etc. After doing this, you should try to generalize your observations and figure out why it's true.
