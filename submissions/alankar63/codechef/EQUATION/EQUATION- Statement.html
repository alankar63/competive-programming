Statement

Input

The first line contains the number of test cases T. Each test case contains 4 integers, N,A,B,C. 0 ? N,A,B,C ? 2500

Output

Output T lines, one for each test case.

Sample Input

4 3 2 1
1 1 1 1

Sample Output
